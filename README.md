# Övningsfiler: Bil och Rektangel

Innehåller exempel på klasser och små program för övningar.

Filer:

- `Bil.py` - klass `Bil` med attribut för registreringsnummer, fabrikat, årsmodell,
  tjänstevikt, motoreffekt och ägare.
- `person.py` - enkel `Person`-klass som används som ägare.
- `bil_program.py` - demo som skapar två bilar med ägare och skriver ut information.
- `rektangel.py` - klass `Rektangel` (höjd och bredd är privata, med get/set-metoder).
- `rektangel_program.py` - interaktivt program som läser höjd/bredd för flera rektanglar
  och skriver ut area och omkrets.

Körning (Windows PowerShell):

python .\bil_program.py
python .\rektangel_program.py

Observera: filerna är avsedda för inlärning och är skrivna på svenska.
